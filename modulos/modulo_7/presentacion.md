# Presentación del Módulo 7: Perspectiva Básica

## Descripción General

Este es un módulo "mágico" donde el papel deja de ser plano y se convierte en una ventana tridimensional. Introducimos la Perspectiva de un Punto de Fuga de forma intuitiva (usando la metáfora del "imán"). Enseñamos que las reglas del dibujo a veces contradicen la intuición plana (por ejemplo, que una carretera es un triángulo). Es un módulo técnico simplificado para niños.

## Objetivos de Aprendizaje (Competencias)

1. **Reducción por Distancia**: Comprender y aplicar la regla de "Lejos = Pequeño / Cerca = Grande".
2. **Punto de Fuga**: Dibujar líneas diagonales que convergen en un solo punto para crear profundidad (caminos, vías, edificios).
3. **Verticalidad**: Mantener las líneas verticales rectas (paredes, postes) incluso cuando hay perspectiva.
4. **Uso del Horizonte**: Colocar elementos correctamente sobre la línea de tierra para generar planos de profundidad.

## Rúbricas de Evaluación Sugeridas

| Criterio | Excelente (10) | Bueno (8) | En Desarrollo (6) |
| :--- | :--- | :--- | :--- |
| **Convergencia (El Imán)** | Las líneas de profundidad (techos, caminos) apuntan claramente al Punto de Fuga. | La mayoría apunta al punto, pero algunas líneas se "escapan" o son paralelas cuando deberían fugar. | Las líneas son paralelas o van en direcciones aleatorias. No hay sensación de profundidad. |
| **Estabilidad Vertical** | Las paredes y postes son verticales (`|`). Los edificios se ven sólidos. | Algunas paredes están inclinadas innecesariamente (efecto terremoto), pero se entiende la intención. | Todas las líneas están chuecas o inclinadas. Los objetos parecen estar cayéndose. |
| **Escala de Profundidad** | Dibuja objetos (árboles, personas) más pequeños a medida que se alejan. | Reduce el tamaño de los objetos lejanos, pero la proporción no es constante (el árbol lejano es muy grande). | Dibuja todo del mismo tamaño sin importar si está cerca o lejos. |

## Recomendaciones Didácticas para el Instructor/Padre

* **Verde y Rojo**: Usa un lápiz rojo para trazar las líneas de fuga (guías) y uno negro para el dibujo final. Ayuda a distinguir la estructura.
* **Fotos de Carreteras**: Muestra fotos reales de carreteras o vías de tren. Pregunta "¿Ves cómo se tocan al fondo?".
* **Regla Obligatoria**: En este módulo, el uso de regla es bienvenido (y necesario) para las líneas rectas de los edificios.
* **No Complicar**: Quédate solo con UN punto de fuga central. Dos puntos es demasiado complejo para esta etapa.
