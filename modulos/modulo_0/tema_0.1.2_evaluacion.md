# Evaluación: Maestro de Materiales

## Pregunta 1

Si quiero pintar una zona muy grande de la hoja rápidamente, ¿qué herramienta es mejor?

- A) Un lápiz de punta muy fina (tardarías años).
- B) Un crayón o marcador grueso.
- C) Una goma.

## Pregunta 2

¿Para qué sirve aprender a controlar la **presión** (fuerza) de la mano?

- A) Para romper el papel y demostrar fuerza.
- B) Para hacer diferentes tonos (claros y oscuros) con un mismo lápiz.
- C) Para cansarnos más rápido.

## Pregunta 3

La Goma de Borrar sirve para corregir errores, pero ¿para qué más la usan los artistas?

- A) Para lanzarla a sus amigos.
- B) Para comerla si tienen hambre.
- C) Para "dibujar" luces y brillos borrando zonas oscuras.

## Pregunta 4

Verdadero o Falso:
"Un buen artista deja todos sus materiales tirados en el suelo al terminar."

- ( ) Verdadero (el desorden es creativo).
- ( ) Falso (hay que cuidar las herramientas para que funcionen bien la próxima vez).
