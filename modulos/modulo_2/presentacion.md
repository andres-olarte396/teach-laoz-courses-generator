# Presentación del Módulo 2: Control de Línea y Trazado

## Descripción General

Este módulo se centra en la "calidad de línea". Ya no basta con hacer una línea; ahora importa *cómo* es esa línea. Exploramos el peso (presión), la velocidad y la intención. Además, introducimos las texturas y patrones como herramientas para rellenar espacios y dar riqueza visual a los dibujos (pasto, madera, escamas) sin necesidad de sombrear todavía.

## Objetivos de Aprendizaje (Competencias)

1. **Modulación de Presión**: Controlar la fuerza de la mano para crear líneas oscuras (fuertes) y claras (suaves/fantasmas).
2. **Variedad de Trazo**: Dibujar líneas con diferentes personalidades (nerviosas, fluidas, quebradas) para expresar emociones.
3. **Generación de Texturas**: Rellenar superficies usando patrones repetitivos (puntos, rayas, tramas) para simular materiales.
4. **Paciencia y Precisión**: Mantener un patrón constante en un área grande sin perder la calidad.

## Rúbricas de Evaluación Sugeridas

| Criterio | Excelente (10) | Bueno (8) | En Desarrollo (6) |
| :--- | :--- | :--- | :--- |
| **Control de Presión (Valores)** | Logra diferencias claras entre líneas fuertes y suaves. Puede hacer degradados de línea. | Se nota la diferencia entre fuerte y suave, pero le cuesta mantener la presión constante en trazos largos. | Todas las líneas tienen la misma presión (generalmente muy fuerte). No logra hacer trazos suaves. |
| **Consistencia de Patrones** | Los patrones (tramas, puntos) son regulares y ordenados. Cubren el área uniformemente. | El patrón se entiende, pero se vuelve desordenado o cambia de tamaño en algunas partes. | El patrón es caótico o se convierte en rayones rápidos por impaciencia. Deja muchos espacios en blanco. |
| **Intención Expresiva** | Usa el tipo de línea correcto para el objeto (suave para nubes, fuerte para piedras). | Intenta variar la línea, aunque a veces usa trazados duros para objetos suaves. | Usa el mismo tipo de línea rígida para todo, sin importar qué objeto sea. |

## Recomendaciones Didácticas para el Instructor/Padre

* **Ejercicios de Calentamiento**: Antes de empezar, hagan "ondas de mar" o "resortes" en una hoja sucia para soltar la muñeca.
* **Lápices Diferentes**: Si es posible, prueben lápices 2B, 4B y 6B para que sientan la diferencia de suavidad.
* **El Juego de la Música**: Pongan música agresiva (rock) y dibujen líneas fuertes; luego música suave y dibujen líneas delicadas.
* **Paciencia con los Puntos**: El puntillismo cansa. Permite descansos breves para sacudir la mano.
