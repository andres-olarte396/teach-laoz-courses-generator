# Configuración del Curso: Dibujo de 0 a Experto para Niños

## Parámetros del Curso

- **TEMA_CURSO**: Dibujo de 0 a Experto para Niños
- **COMPLEJIDAD**: Baja → Alta (progresiva)
- **DURACIÓN**: 100+ horas
- **AUDIENCIA**: Niños y niñas de 6 años en adelante
- **PRERREQUISITOS**: Ninguno (curso desde cero)
- **ENFOQUE**: Dibujo libre y creatividad
